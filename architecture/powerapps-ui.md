# Power Apps UI Architecture

This document defines the user interface structure of the Productivity Platform.

---

# Main Screens

## Home Dashboard

Purpose:

Central operational overview.

Components:

- Tasks due today
- Pending priorities
- Project overview
- Alerts

---

## Tasks

Purpose:

Manage operational tasks.

Features:

- Create task
- Edit task
- Assign responsible
- Set priority
- Set due date
- Link to project

Views:

- My tasks
- All tasks
- Overdue tasks

---

## Projects

Purpose:

Manage projects and strategic initiatives.

Features:

- Create project
- Link tasks
- Track project status
- Timeline view

---

## Kanban Board

Purpose:

Visual workflow management.

Columns:

- Pending
- In Progress
- Blocked
- Completed

Features:

Drag & drop task movement.

---

## Daily Scrum

Purpose:

Register daily operational meetings.

Features:

- Daily notes
- Pending tasks
- Carry over unfinished tasks

---

## Operational Dashboard

Purpose:

Productivity monitoring.

Metrics:

- Tasks completed
- Overdue tasks
- Active projects
- Productivity trends

---

## Portfolio

Purpose:

Store validated projects and product cases.

Content:

- Case documentation
- Validation records
- Impact metrics
