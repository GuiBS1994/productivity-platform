# Productivity Platform

Internal productivity and operations management platform built with Power Platform.

## Objective

Create a centralized system to manage tasks, projects, priorities and operational workflows.

## Main Modules

- Task Management
- Project Management
- Kanban Board
- Daily Scrum
- Email Integration
- Operational Dashboard
- Project Portfolio

## Technology Stack

- Power Apps
- Power Automate
- SharePoint
- Outlook

## Development Method

Agile development with modular architecture.
Each module will be developed and validated independently.
